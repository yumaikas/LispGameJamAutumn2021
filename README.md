# AutumnLispGameJam

## Up and running

Install [Janet](https://github.com/janet-lang/janet/releases), and then [JPM](https://github.com/janet-lang/jpm)

Then run `jpm deps` to install jaylib, and run `janet AutumnLispGameJam.janet` to run/test the game
